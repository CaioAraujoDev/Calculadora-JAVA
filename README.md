# Calculadora em Java

Projeto de calculadora simples em java, realizando operações básicas como soma, subtração, multiplicação e divisão.

## Status
Concluído

## Funcionalidades
- Soma 2 números
- Subtrai 2 números
- Multiplica 2 números
- Divide 2 números
- Trata erros de entrada inválida

## Tecnologias utilizadas
- Java
- IntelliJ IDEA
- Git e Github

## Como executar
1. Instale o Java(versão 17 ou superior)
2. Clone ou baixe este repositório:  
git clone https://github.com/CaioAraujoDev/Calculadora-JAVA.git
3. Abra o projeto no seu IDE(recomendado: Intellij IDEA)
4. Execute a classe principal(Main)
## Exemplo de uso
Qual sera sua operação(+,-,*,/): +  
Digite o primeiro número: 10  
Digite o segundo número: 20  
Resultado: 30  
Desejar fazer outro cálculo ?(S/N): N
